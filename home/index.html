What is Dijkstra's algorithm?

It is an algorithm, invented by Dijkstra, which finds the shortest path from a given source node to every other node.

Its optimized version makes the algorithm asymptotically the fastest known single-source shortest-path algorithm for arbitrary directed graphs with unbounded non-negative weights.

Time complexity

The time complexity of Dijkstra's algorithm depends mainly on the data structure used to represent the graph.

How does it work?

Given a graph with several nodes, we have to use Dijkstra's algorithm to find the minimum spanning tree. Let's start by checking every distance between the starting node and its adjacent nodes. After finding the minimum one, let's repeat the same process in this adjacent node. By repeating this process each time, we will be able to find the minimum path which connects every node.
